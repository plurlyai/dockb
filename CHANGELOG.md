## [0.1.0] - 2026-03-15

### Added

- Initial README with project purpose, repository structure, and agent design guidelines.
- `.agent.md` agent registration with tool bindings and file-type patterns (`.md`, `.mdx`, `.rst`, `.txt`, `.adoc`).
- `docs.instructions.md` with agent behavior rules, guardrails, workflows, and output style.
- `docs.skill.md` with documentation capabilities, style rules, and common actions.
- `docs.prompt.md` with reusable prompt templates for README generation, feature docs, QA, and changelog updates.
- `docs.changelog.md` template for changelog generation guidance.
- Comprehensive setup guide for GitHub Copilot + VS Code integration in README.
- Versioning and changelog guidance across instructions, skill, and prompt files.
